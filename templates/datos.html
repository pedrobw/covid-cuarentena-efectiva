Información sobre los datos

Fuente de los datos

Todos los datos aquí utilizados se encuentran abiertamente disponibles en el sitio del Ministerio de Ciencia, Tecnología, Conocimiento e Innovación, o más precisamente, en su GitHub abierto. De ellos, se escogieron principalmente los datos específicos a las comunas, es decir, los ahí descritos como Datos por comuna, pero también, los viajes diarios por comuna en transporte público en la sección de Movilidad.

Objetivo del proyecto

Este proyecto fue desarrollado en Flask por mí, Pedro Bahamondes, para el Instituto Milenio Fundamentos de los Datos (para el cual actualmente investigo para mi tesis de magíster) en el contexto de una postulación laboral. Toda el código fuente de este proyecto está disponible de manera abierta en mi GitHub personal.
